# 05 Agents

Run Climpt agents for automated iteration and review workflows.

## Scripts

| Script               | Description                              |
| -------------------- | ---------------------------------------- |
| `01_run_iterator.sh` | Run the iterator agent on a GitHub issue |
| `02_run_reviewer.sh` | Run the reviewer agent on a project      |
| `03_agent_config.sh` | Show agent configuration structure       |

## Prerequisites

- Climpt installed and initialized
- GitHub CLI (`gh`) authenticated (for issue-based agents)
