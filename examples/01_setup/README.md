# 01 Setup

Install Climpt and initialize the project directory.

## Scripts

| Script               | Description                                              |
| -------------------- | -------------------------------------------------------- |
| `01_install.sh`      | Install Climpt globally from JSR                         |
| `02_init_basic.sh`   | Run `climpt init` to create `.agent/climpt/`             |
| `03_init_options.sh` | Demonstrate init options: `--force`, `--skip-meta`, etc. |

## Prerequisites

- Deno 2.x installed
