# 03 MCP Server

Start the Climpt MCP server and integrate with AI-powered editors.

## Scripts

| Script                     | Description                           |
| -------------------------- | ------------------------------------- |
| `01_start_server.sh`       | Start the MCP server                  |
| `02_claude_integration.sh` | Configure Claude / Cursor integration |

## Existing Config Files

| File                      | Description                      |
| ------------------------- | -------------------------------- |
| `mcp-config.example.json` | Example MCP server configuration |
| `registry.template.json`  | Template for command registry    |

## Prerequisites

- Climpt installed and initialized
- Claude Desktop or Cursor (for IDE integration)
