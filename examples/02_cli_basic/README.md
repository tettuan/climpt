# 02 CLI Basic Usage

Core CLI commands for decomposition, summarization, and defect analysis.

## Scripts

| Script                   | Description                            |
| ------------------------ | -------------------------------------- |
| `01_decompose.sh`        | Decompose: project, issue, task        |
| `02_summary.sh`          | Summarize scattered information        |
| `03_defect.sh`           | Analyze defects from error logs        |
| `04_stdin_input.sh`      | Pipe input via STDIN                   |
| `05_custom_variables.sh` | Pass user-defined variables (`--uv-*`) |

## Prerequisites

- Climpt installed and initialized (`climpt init`)
