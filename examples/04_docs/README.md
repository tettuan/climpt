# 04 Documentation Installer

List, install, and filter Climpt documentation packages.

## Scripts

| Script               | Description                           |
| -------------------- | ------------------------------------- |
| `01_list_docs.sh`    | List available documentation packages |
| `02_install_docs.sh` | Install docs into a local directory   |
| `03_filter_docs.sh`  | Filter by category, language, or mode |

## Prerequisites

- Deno 2.x installed
